# gray-inversion
本项目可以用来对单张图片以及对文件夹进行灰度反转的操作

Gray_Inversion_Onlyimage主要是对单张图片进行的操作
而Gray_Inversion_folder主要是对文件夹（包含大量文件）进行的操作

主要用到opencv，而在处理文件夹的时候用到了python自带的操作系统os，所以会有import os的开头

在处理文件夹的时候我注释掉了一些语句，比如只处理文件夹里面部分类型的图片那句，因为我确定我自己的文件夹里面都是同一种类型的图片，但是在处理未知文件夹的时候最好还是带上，避免程序崩溃。
还有就是在代码里面我注释掉了创建新的输出文件夹这句代码，因为在程序运行之前我已经新建了输出的文件夹，所以之后如果要再次跑这个程序的时候要么需要自己手动创建新的文件夹，要么用程序新建文件夹

Github上我没有上传测试的图集，可能需要自己去找数据集进行测试

在整个灰度操作的过程中，我用到的是绝对地址，就是从C盘开始一直到文件夹的地址作为输入和输出，也可以用到相对地址，比如项目地址是这样的：
IMAGE/
  Gray Inversion.py
  images/
    test.jpg
那就可以写img = cv2.imread("images/test.jpg")
如果觉得麻烦，就用绝对地址，记得绝对地址的斜杠要是/而不是\，一般从文件夹上面复制粘贴过来的是\，我们需要手动改成/，或者用r:"C\..."
